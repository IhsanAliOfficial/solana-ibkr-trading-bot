solana-ibkr-bot (LIVE-ready version)

What this is

A production-ready Python trading bot for Solana (SOL/USD) using the Interactive Brokers (IBKR) TWS API.
It connects directly to your IB Gateway or TWS client and executes real market orders based on your custom strategies (default = SMA crossover).
Includes a minimal PySimpleGUI dashboard and optional live price chart.
